# Sales Data Cleaning & Analysis Using SQL

## Project Overview  
This project focuses on cleaning and analyzing a raw sales dataset using SQL. The dataset contained inconsistencies such as duplicate records, missing values, and inconsistent text formatting.

---

##  Data Cleaning Process  
- Standardized text using UPPER()  
- Handled missing values using IFNULL()  
- Removed duplicate records using DISTINCT  

---

##  Data Analysis  
- Total sales per city  
- Best performing product  
- Customer distribution  

---

##  Key Insights  
- Data cleaning improved accuracy  
- Laptop generated the highest revenue  
- Sales varied across cities

- SQL
